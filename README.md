## <a name="introduction">✨ Introdução</a>

AI-powered modern stock market app built with Next.js, Shadcn, Better Auth, and Inngest! Track real-time prices, set personalized alerts, explore company insights, and manage watchlists. The admin dashboard allows managing stocks, publishing news, and monitoring user activity, while event-driven workflows power automated alerts, AI-driven daily digests, earnings notifications, and sentiment analysis—perfect for devs who want a dynamic, real-time financial platform.

Se você estiver começando, precisar de assistência ou encontrar algum _bug_, fique à vontade para abrir uma _Issue_ neste repositório.

## <a name="tech-stack">⚙️ Tech Stack</a>

- **[Better Auth](https://www.better-auth.com/)** é uma biblioteca de autenticação e autorização agnóstica a frameworks para TypeScript. Ela fornece suporte integrado para login por e-mail/senha, _social sign-on_ (Google, GitHub, Apple e mais) e autenticação multifator, simplificando a autenticação de usuários e o gerenciamento de contas.

- **[CodeRabbit](https://www.coderabbit.ai/)** é um assistente de revisão de código baseado em IA que se integra ao GitHub. Ele ajuda desenvolvedores a capturar _bugs_, reforçar as melhores práticas e manter a qualidade consistente do código em _pull requests_, reduzindo o esforço de revisão manual e acelerando o fluxo de trabalho de desenvolvimento.

- **[Finnhub](https://finnhub.io/)** é uma API de dados financeiros em tempo real que fornece dados de mercado de ações, _forex_ e criptomoedas. Oferece aos desenvolvedores acesso a dados fundamentais, indicadores econômicos e notícias.

- **[Inngest](https://www.inngest.com/)** é uma plataforma para _workflows_ orientados a eventos e trabalhos em segundo plano. Permite que os desenvolvedores criem processos automatizados confiáveis e escaláveis, como alertas em tempo real, notificações e _workflows_ baseados em IA.

- **[MongoDB](https://www.mongodb.com/)** é um banco de dados NoSQL flexível e de alto desempenho. Ele armazena dados em documentos semelhantes a JSON, suporta esquemas dinâmicos e oferece recursos robustos para escalabilidade, replicação e consulta.

- **[Nodemailer](https://nodemailer.com/)** é uma biblioteca Node.js para envio fácil de e-mails. Suporta vários métodos de transporte, como SMTP, OAuth2 e serviços de terceiros, tornando-o uma ferramenta confiável para lidar com e-mails transacionais e notificações em aplicações.

- **[Next.js](https://nextjs.org/docs)** é um poderoso _framework_ React para construção de aplicações web _full-stack_. Ele fornece renderização do lado do servidor, geração de site estático e rotas de API, permitindo que os desenvolvedores criem aplicativos otimizados e escaláveis rapidamente.

- **[Shadcn](https://ui.shadcn.com/docs)** é uma biblioteca de código aberto de componentes React totalmente personalizáveis e acessíveis. Ajuda as equipes a construir UIs consistentes e visualmente atraentes rapidamente, permitindo controle total sobre o design e o layout.

- **[TailwindCSS](https://tailwindcss.com/)** é um _framework_ CSS _utility-first_ que permite aos desenvolvedores construir designs responsivos e personalizados rapidamente, sem sair do HTML.

- **[TypeScript](https://www.typescriptlang.org/)** é um superconjunto de JavaScript com tipagem estática que melhora a qualidade do código, as ferramentas e a detecção de erros. É ideal para construir aplicações em larga escala e aumenta a manutenibilidade.

## <a name="features">🔋 Funcionalidades</a>

👉 **Stock Dashboard**: Rastreie preços de ações em tempo real com gráficos interativos de linha e _candlestick_, incluindo dados históricos, e filtre ações por setor, desempenho ou valor de mercado.

👉 **Powerful Search**: Encontre rapidamente as melhores ações com um sistema de busca inteligente que ajuda a navegar pela plataforma.

👉 **Watchlist & Alerts**: Crie uma _watchlist_ personalizada, defina limites de alerta para mudanças de preço ou picos de volume e receba notificações instantâneas por e-mail para se manter atualizado.

👉 **Company Insights**: Explore dados financeiros detalhados, como índice P/L, EPS, receita, notícias recentes, registros, classificações de analistas e pontuações de sentimento para uma tomada de decisão informada.

👉 **Real-Time Workflows**: Alimentado pelo **Inngest**, automatize processos orientados a eventos, como atualizações de preços, agendamento de alertas, relatórios automatizados e _insights_ baseados em IA.

👉 **AI-Powered Alerts & Summaries**: Gere resumos de mercado personalizados, _digests_ diários e notificações de relatórios de lucros, ajudando os usuários a rastrear o desempenho e tomar decisões baseadas em dados.

👉 **Customizable Notifications**: Ajuste alertas e notificações com base nas _watchlists_ e preferências do usuário para uma experiência altamente personalizada.

👉 **Analytics & Insights**: Obtenha _insights_ sobre o comportamento do usuário, tendências de ações e métricas de engajamento, permitindo decisões de negócios e negociação mais inteligentes.

E muito mais, incluindo arquitetura de código e reusabilidade.
